# PowerPack
A PowerShell module that contains an assortment of functions that can be used for AD enumeration and exploitation.

Functions in this module are:
* Invoke-ShareHunter
* Invoke-SessionHunter
* Invoke-SPNHunter
* Invoke-ACEHunter
* Invoke-TGSGatherer
* Invoke-InfinityHopper 
* Invoke-TrustHunter
* Invoke-DomainHunter
